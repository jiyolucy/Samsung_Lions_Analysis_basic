# 구장별 최적 2루수 선발 전략

### 🧢 류지혁 vs 전병우 vs 심재훈
- 류지혁선수 23년 7월입단, 전병우선수 23년 11월 입단, 심재훈 선수 25년 입단.
- 따라서 세 선수의 공통 활동기간인 **2025년** 데이터를 기준으로 분석
- KBO기록실과 STATIZ기록실의 데이터를 참고해 구장별 최적 선발 2루수 선정

<br>

### 📀 참고 데이터 (Data Check)
- KBO 기록실 (https://www.koreabaseball.com)
- STATIZ 선수정보 (https://www.statiz.co.kr/player)

### 📊 분석 지표 정의 (Data Glossary)
| 항목 | 약어 | 설명 | 비고 |
| :--- | :---: | :--- | :--- |
| **선수 이름** | Name | 분석 대상 선수명 | 류지혁, 전병우, 심재훈 |
| **경기** | G | 해당 시즌 출장 경기 수 | |
| **타석** | PA | 타석에 들어선 총 횟수 | |
| **타율** | AVG | 안타 / 타수 | 타격의 정확도 지표 |
| **안타** | H | 총 안타 개수 | |
| **2루타** | 2B | 총 2루타 개수 | 중장타 생산력 확인 |
| **홈런** | HR | 총 홈런 개수 | |
| **장타율** | SLG | 루타수 / 타수 | 구장 크기 분석의 핵심 지표 |
| **수비관련 득점 기여도** | RAA | 수비 범위, 실점 억제력 | 넓은 수비범위, 정확한 송구 |
| **타자 생산성** | OPS | 출루율 + 장타율 | 구장별 타격 공헌도 판단 기준 |
