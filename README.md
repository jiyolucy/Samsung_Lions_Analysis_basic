## Project Overview
This project analyzes the homerun performance of the SAMSUNG LIONS to compare the difference between HOME and AWAY games.

## Tools Used
- Date Source : Statiz(KBO Statistics)
- Data Cleaning : Excel, Pandas
- Visualization : Python
- Database : MySQL

## Insight
- The LIONS PARK effect is significant homeruns at home are consistently higher than in away games
- 2024 was a peak year for homerun production.
